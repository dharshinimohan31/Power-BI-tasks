**# Task 2 – Gender-based Investment Analysis

## Objective
To analyze and visualize gender-based differences in investment preferences using Power BI.

## Dataset
- Provided by Cognify Technologies
- File: Data_set_2.csv

## Tools Used
- Power BI
- GitHub

## Visualizations
- Clustered Column Chart comparing equity, mutual funds, and government bonds by gender
- Stacked Column Chart showing overall investment distribution

## Key Insights
- Male respondents show higher participation in equity investments.
- Female respondents prefer mutual funds and government bonds, indicating lower risk preference.
- Gender significantly influences investment choices.
